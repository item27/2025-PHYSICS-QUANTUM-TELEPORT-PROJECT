# Математика и вычисления в визуализаторе

Эта папка описывает **как именно считаются значения**, которые вы видите в интерфейсе. Здесь нет сложных формул — только простые объяснения и короткие фрагменты кода для наглядности.

## Что внутри

- `01_bloch_state.md` — как задаём состояние кубита через углы и радиус.
- `02_demo_transitions.md` — как демо-режим двигает углы между шагами.
- `03_server_transitions.md` — какие вычисления выполняет сервер при переходах шагов.
- `04_bloch_projection.md` — как из углов получаем 2D-координаты на сфере Блоха.

## Зачем это нужно

Эти вычисления — **визуальная метафора**, но они остаются согласованными между демо и сетью. Документы помогают быстро понять, где именно в коде задаётся поведение и почему оно выглядит именно так.
